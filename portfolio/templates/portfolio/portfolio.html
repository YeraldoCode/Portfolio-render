Econome

Aplicacion web para la gestion de finanzas personales incluye un login para que el usuario pueda acceder a su cuenta y un dashboard para ver su estado financiero. formulario para agregar ingresos y gastos, y un grafico para ver el progreso de las finanzas. se programo con python y flask, las librerias usadas son sqlite3, sqlalchemy, y flask_login se utilizo bootstrap para el diseño de la pagina.

InfoBag

Aplicacion de escritorio donde se aplica la tecnica del web scraping para obtener informacion de las paginas web. esta aplicacion obtiene informacion de las empresas que cotizan en la bolsa de valores, para ello se utiliza la API de Yahoo Finance. Al agregar el icono de la empresa en la barra de busqueda, se obtiene la informacion de la empresa y se muestra en una ventana emergente. Se programo con python y tkinter, las librerias usadas son beautifulsoup4, requests, pandas, y yahoo_fin.
